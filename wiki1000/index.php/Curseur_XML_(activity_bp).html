Curseur XML (activity bp)

De Wiki1000
Aller à : Navigation, rechercher

Cette activité permet d'exécuter un curseur sur un document XML.

  • Un pas du curseur est un élément fils de l'élément racine du document.
  • A chaque pas du curseur, donc pour un élément fils de l'élément racine, le curseur crée un paramètre par élément fils.

Par exemple, pour un document ayant la structure suivante : 

<items>
 <item>
   <el1>value11</elt1>
   <el2>value12</elt2>
   ...
 <item>
 <item>
   <el1>value21</elt1>
   <el2>value22</elt2>
   ...
 <item>
<items>

le pas du curseur correspond aux éléments "item" et pour chaque pas le curseur crée les paramètres prefixe.el1 et prefixe.el2 avec pour valeur les valeurs des éléments.

Onglet Propriétés

  • Fichier
Une référence sur le document XML à parcourir.
  • Préfixe des paramètres
Le préfixe des paramètres qui seront créés dans le contexte de l'instance.
