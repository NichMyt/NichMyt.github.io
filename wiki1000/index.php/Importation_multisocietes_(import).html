Importation multisocietes (import)

De Wiki1000
Aller à : Navigation, rechercher

Il est possible de paramétrer une importation pour l'exécuter dans plusieurs sociétés d'un dossier mult-sociétés.

Les prés requis sont :

  • Configurer un dossier multi-sociétés.
  • Configurer un service sur ce dossier.

Deux approches sont possibles :

  • Utilisation d'un seul fichier contenant toutes les données.
Dans ce cas le descriptif d'importation filtre les données à importer pour chaque société.
  • Utilisation d'un fichier par société.
Dans ce cas le nom du fichier à importer est paramétré dans le descriptif d'importation.
Tip : Ce type d'importation enchaîne sur chaque société définies dans la tâche et filtre les données d'importation dans chacun des contextes sociétés.
Attention : Certain processus d'importation utilisent un objet paramétrage d'importation qui est de niveau société. Dans ce cas, vous devez créer un paramètre d'importation dans chacune des sociétés concernées avec le même code de paramètre de sorte que le processus retrouve le paramétrage pour chaque société.

La configuration se fait en trois étapes :

1. Ajouter un critère au descriptif d'importation pour filter les données par société :

2. Planifiez ensuite l'importation dans un automate.

3. Configurez ensuite la tâche d'automate pour quelle s'exécute sur toutes les bases du dossier :

Vous pouvez :

  • Sélectionner toutes les Sociétés du Dossier en décochant toutes les bases de données.
  • Sélectionner uniquement certaines Sociétés du Dossier en cochant les bases concernées.

Utilisation d'un fichier de données par société.

1. Lors de la configuration de l'import paramétrez le nom du fichier pour intégrer une section comportant le nom de la base de données :

2. Planifiez ensuite l'importation dans un automate.

3. Configurez ensuite la tâche d'automate pour quelle s'exécute sur toutes les bases du dossier :

Vous pouvez :

  • Sélectionner toutes les Sociétés du Dossier en décochant toutes les bases de données.
  • Sélectionner uniquement certaines Sociétés du Dossier en cochant les bases concernées.
