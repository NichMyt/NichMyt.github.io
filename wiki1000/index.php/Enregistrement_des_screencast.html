Enregistrement de vidéo

Les outils utilisés :

  • CamStudio
CamStudio est un freeware qui permet l'enregistrement de captures d'écran sous forme de vidéo
  • ffmeg
ffmeg est un utilitaire en ligne de commande qui permet de convertir une vidéo dans le format Flash flv.
  • VirtualDub
VirtualDub est un éditeur de vidéo (AVI) freeware.

Paramétrage de CamStudio

  • Sélectionner le codec "CamStudio loosless codec" ; ce codec permet l'enregistrement des captures d'écran sans perte de qualité.
  • Paramétrer une touche de raccourci pour débuter l'enregistrement et une autre pour l'arrêter.

Paramétrage du bureau et de l'application

  • Paramétrer le bureau Windows sur la résoution 1024x768.
  • Masquer les icônes de bureau pour avoir un bureau propre.
  • Utiliser l'application 1000 standard.
  • Sélectionner le style par défaut.

Edition des vidéos

VirtualDub peut être utilisé pour monter les vidéos et assembler plusieurs vidéos en une seule.

Conversion des vidéos

Après enregistrement il est nécessaire de convertir la vidéo dans le format flash, pour cela exécuter ffmeg dans le répertoire ou est stockée la vidéo :

convert.bat ffmpeg -i %1.avi -qscale 10 -vcodec libx264 %1.flv


  • Avec la dernière version de ffmpeg il est préférable d'utiliser ces options (08/2013) 
 convert.bat ffmpeg -i %1.avi -qscale 10 -vcodec libx264 -pix_fmt yuv420p %1.flv

puis convert ma-video

Publication des vidéos sur le wiki

  • Nommer la video correctement en la préfixant du numéro de version correspondant, par exemple : 
v560-dsm-xxxx.flv
  • Copier la video flv obtenue dans un sous répertoire de

\\l1000-portal\wikipubvideos Dans la page wiki utiliser la syntaxe suivante : 

{{#flashs:ma-video.flv|mon-repertoire|Titre de ma video}}
Tip : Ne pas utiliser d'accent ni d'apostrophe dans le titre de la video
