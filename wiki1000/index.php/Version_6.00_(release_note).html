Version 6.00

Cette page résume les modifications apportées dans le version 6.00.

Application Desktop

Style personnalisable

L'application Desktop utilise un nouveau Visuel apportant un plus grand confort d'utilisation.

Bureau en mode onglet

Le mode Bureau en onglet permet d'afficher les écrans dans des onglets séparés.

  • Gestion de Marque-pages (Bookmark) en menu et en barre générale.
  • Gestion multi-bureaux

Grille

  • Mise en évidence des zones actives
  • Accès directe au menu de propriété d’une colonne
  • Menus de configuration de la position des colonnes
  • Filtre sur les valeurs des colonnes

Nouvelles options sur les regroupements :

  • Option pour qu'un groupe "Toujours déplié" ne soit pas affiché
  • Option pour qu'un groupe "Toujours replié" soit affiché comme une ligne de détail
  • Option pour afficher des lignes cumuls de regroupement

Gestion des devises :

  • Prise en compte de la devise pour les colonnes montants.
  • Afficher une ligne par devise dans les totalisations de cette colonne.

Export de la grille

  • Support de OpenDocument 1.2
Ce format a été testé avec les produits suivant :
Open Office 3
Microsoft Office 2007 SP2
Google Docs
  • Modification de l'export XML pour un format plus facilement exploitable.
  • Ajout d'une option pour envoyer le document exporté en Mail

Menus riches

  • Gestion des Icônes de menu par type de fonctionnalité
  • Affichage des informations enrichies dans les éléments de menu
  • Affiche de libellé pour les séparateurs
  • Séparateur avec plier / déplier des éléments contenus
  • Sauvegarde de l’état plier / déplier par utilisateur

Barre Outlook

  • Nouveau design de la barre Outlook
  • Réorganisation du menu contextuel de la barre
  • Nouveau panneau Style
  • Nouveau panneau Sélecteur de partage.

Combobox et recherche

  • Grille multi-colonne en remplacement de la liste.
  • Sauvegarde des positions
  • Sélection avec accès rapide en panneau.
  • Liste des derniers éléments utilisés en panneau.
  • Tri sur toutes les colonnes.
  • Recherche incrémentale sur toutes les colonnes.
  • Recherche sur les dates

Treebook et panneaux

  • Ajout de panneau dans les Treebooks des écrans
  • Panneau des dernières actions réalisées
  • Panneau des Sélections de l'utilisateur
  • Panneau des documents associés à l'objet.

Préférences utilisateurs

  • Amélioration du support des préférences utilisateurs
  • Stockage des préférences par Dossier et par contexte Société

Application Web

Ergonomie et productivité

  • Reprise des amélioration d'ergonomie de l'application Desktop.
  • Amélioration du gestionnaire de fenêtre.
  • Nouvelle grille de la grille de saisie et de consultation.
  • Meilleur support du clavier et des raccourcis.

Style personnalisable

Support de plusieurs Style (CSS) personnalisables par utilisateur.

Applications Outlook

Les applications Outlook sont des extension d'Applications Métiers intégrées dans la Suite Microsoft Office et utilisant Outlook comme logiciel client.

Les applications Outlook sont constituées par :

  • Un Add-On Outlook générique installé sur chaque poste utilisateur.
  • Des Applications Métiers s'exécutant dans Outlook permettant à un utilisateur d'effectuer certaines tâches métiers.

Les Applications Outlook ont les caractéristiques suivantes :

  • Elles ciblent un aspect métier précis d'un utilisateur.
  • Elles fonctionnent sur un mode de Licences nommées.

Portail

  • Nouveau Visuel de vignette apportant un meilleur confort d'utilisation.
  • Possibilité d'ajouter des informations complémentaires dans les Vignettes RSS
  • Possibilité d'exporter les onglets d'un portail utilisateur et de les importer dans le portail d'un autre utilisateur.
  • Nouveau type de vignette métier.

Administration

  • Nouveau modèle d'authentification permettant une gestion arborescente des entités.
  • Modèle d'authentification Multi-Annuaire d'Entreprise.
  • Support des annuaires LDAP et Active Directory.
  • Support des Dossiers Mono-Base Multi-Société.
  • Support du Copier/Coller dans l'arbre de l'administrateur en supplément de la fonction Glisser/Déposer.
  • Support de l'historisation des objets du référentiel.

Architecture multi-sociétés (partage)

Le framework 6.00 implémente le partage des objets métiers entre différents périmètres.

Langage

  • Opérations de classe
  • Opérateurs As et Is
  • Opérateur ForEach
  • Opérateur Case sur les expressions entières et chaînes
  • Amélioration du support de Assign()
  • Amélioration du support de _TP()
  • Amélioration du contrôle de type
  • Typage de la librairie d'exécution (RTL)

Traduction

  • Amélioration du support de la traduction.

Outils de développement

  • Remplacement de la documentation de développement par ce wiki.
  • Amélioration de la recherche incrémentale dans les classes.
  • Ajout de la fonction "pas à pas approfondi" (F7) dans le débugeur.
  • Amélioration du CodeInsight pour une meilleur prise en compte des objets techniques
  • Ajout d'une fonction de trace des modifications d'attributs

Automate

  • Possibilité d'associer plusieurs Sociétés d'un Dossier à une tâche d'automate pour qu'elle s'exécute sur chaque Société du Dossier.
  • Possibilité d'associer plusieurs Dossiers (bases de données) à une tâche d'automate pour qu'elle s'exécute sur chaque Dossier.
  • Support de SNMP (Automate exécuté en Service)

Processus métier et Workflow

  • Support de SNMP, les processus et les activité peuvent générer des traps SNMP.
  • Gestion des évènements métiers par file d'attente
  • Activités pour gérer des requêtes HTTP
  • Activités HTTP GET et POST.
  • Activités pour gérer des files d'attente MSMQ
  • Activité pour envoyer des messages MSMQ.
  • Evènement initial pour lire une file d'attente MSMQ.
  • Activités pour manipuler des documents XML
  • Requête XML
  • Curseur XML
  • Transformation XSLT
  • Expression XPath
  • Validation par un schéma XML

Publication de documents

La publication permet d'automatiser l'édition et la diffusion d'états Ligne 1000.

Voir aussi :

Publication de document

Import / Export

  • Support des dossiers Multi-Sociétés
  • Possibilité de paramétrer les requêtes d'importation SQL en fonction du contexte d'exécution.
  • Possibilité de déplacer le fichier source après importation.
  • Possibilité d'envoyer un mail si succès ou échec après importation.
  • Possibilité de réaliser des importations multi-bases

Web Services

  • Nouvelle implémentation de la couche SOAP s'appuyant sur les Schémas XML.
  • Possibilité de publier une requête Ligne 1000 sous forme de Web Service.
  • Possiblité d'authentifier un Web Service.

Voir aussi :

Migration des Web Services en version 6.00

Serveur

  • Optimisation du serveur pour la montée en charge.
  • Support des processeurs Multi-coeurs.
  • Gestion de l'affinité du Processus.
  • Support du protocole SNMP et des TrapV1 SNMP.
  • Possibilité de déléguer l'exécution des tâches d'automate sur un autre service.

Voir aussi :

Mise en oeuvre des traps SNMP.

Requêteur

  • Requêtes ETL
  • Multi-sociétés
  • Gestion des enumérés
  • Gestion des paramètres de type liste
  • Gestion de la traduction
  • Flux RSS
  • Publication (Services web)
  • Gestion des dates relatives
  • Colonne nom du fichier d'exportation

Support

La version Sage Ligne 1000 6.00 et Sage Ligne 1000 6.00 Edition Piloté supporte les produits tiers suivant :

Stations Desktop :

  • Windows XP 32 bits
  • Windows Vista 32 bits
  • Windows Vista 64 bits
  • Windows Seven 32 bits
  • Windows Seven 64 bits

Serveurs :

  • Windows Server 2003 32 Bits
  • Windows Server 2003 64 Bits (*)
  • Windows Server 2008 32 Bits
  • Windows Server 2008 64 bits (*)

(*) Un paramétrages correct de l'OS et de IIS doivent être effectuées.

Serveurs de données :

  • Microsoft SQL Server 2005
  • Microsoft SQL Server 2008
  • Oracle 10g Rx
  • Oracle 11g
  • Sage SQL

Serveurs HTTP :

  • IIS 6.x, 7.x
  • Apache Server 1.3x, 2.0x, 2.2x

Navigateurs :

  • Mozilla Firefox 3.x 3.5x
  • Internet Explorer IE7 et IE8

Serveurs décisionnel :

  • Business Object XI 3.1


Les versions de produit suivants sont supportées mais dépréciées, il est conseillé de migrer vers une version supérieure :

  • Windows 2000
Arrêt du support par Microsoft, nécessite d'installer le GDI+
  • Microsoft Server 2000
Arrêt du support par Microsoft
  • IIS 5.x
Version de IIS sur Microsoft Server 2000
  • Microsoft SQL 2000
Meilleurs performances sur les versions supérieures.
  • Oracle 9x
Meilleurs performances sur les versions supérieures.
  • Business Object XI 3.0
Différents problèmes ont été corrigés par BO sur la version 3.1

Les versions de produit suivants ne sont plus supportées :

  • Internet Explorer IE6x
Problème d'affichage sur les images png
